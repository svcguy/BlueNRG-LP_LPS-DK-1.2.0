Detailed Description

How to use the TIM peripheral to measure an external signal frequency.

KEIL project

To use the project with KEIL uVision 5 for ARM, please follow the instructions below:

  1. Open the KEIL uVision 5 for ARM and select Project->Open Project menu.
  2. Open the KEIL project C:\Users\{username}\ST\BlueNRG-LP_LPS DK x.x.x\Projects\Periph_Examples\HAL\TIM\TIM_InputCapture\MDK-ARM\{STEVAL-IDB011V1|STEVAL-IDB012V1}\TIM_InputCapture.uvprojx
  3. Select desired configuration to build
  4. Select Project->Rebuild all target files. This will recompile and link the entire application
  5. To download the binary image, please connect an USB cable in your board (CMSIS-DAP upgrade).
  6. Select Project->Download to download the related binary image.
  7. Alternatively, open the BlueNRG-LP Flasher utility and download the built binary image.

IAR project

To use the project with IAR Embedded Workbench for ARM, please follow the instructions below:

  1. Open the Embedded Workbench for ARM and select File->Open->Workspace menu.
  2. Open the IAR project C:\Users\{username}\ST\BlueNRG-LP_LPS DK x.x.x\Projects\Periph_Examples\HAL\TIM\TIM_InputCapture\EWARM\{STEVAL-IDB011V1|STEVAL-IDB012V1}\TIM_InputCapture.eww
  3. Select desired configuration to build
  4. Select Project->Rebuild All. This will recompile and link the entire application
  5. To download the binary image, please connect an USB cable in your board (CMSIS-DAP upgrade).
  6. Select Project->Download and Debug to download the related binary image.
  7. Alternatively, open the Flasher utility and download the built binary image.

WiSE-Studio project

To use the project with WiSE-Studio IDE (GCC toolchain), please follow the instructions below:

  1. Open the WiSE-Studio IDE
  2. Select File, Import, Existing Projects into Workspace C:\Users\{username}\ST\BlueNRG-LP_LPS DK x.x.x\Projects\Periph_Examples\HAL\TIM\TIM_InputCapture\WiSE-Studio\{STEVAL-IDB011V1|STEVAL-IDB012V1}
  3. Select desired configuration to build
  4. Select Project->Build Project. This will recompile and link the entire application
  5. To download the binary image, please connect an USB cable in your board (CMSIS-DAP upgrade).
  6. Select Run->Run/Debug to download the related binary image.
  7. Alternatively, open the Flasher utility and download the built binary image.

Usage

How to use the TIM peripheral to measure an external signal frequency.

The TIM1CLK frequency is set to 64MHz, the Prescaler is 0, so the TIM1 counter clock is 64MHz.

TIM1 is configured in Input Capture Mode: the external signal is connected to TIM1 Channel1 used as input pin. To measure the frequency we use the TIM1 interrupt request, so in the TIM1 IRQHandler routine, the frequency of the external signal is computed.

The "uwFrequency" variable contains the external signal frequency: uwFrequency = TIM1 counter clock / uwDiffCapture (Hz), where "uwDiffCapture" is the difference between two consecutive TIM1 captures.

The minimum frequency value to measure is TIM1 counter clock / CCR MAX = 64 MHz / 65535

the external signal must be connected to TIMx Channelx used as input pin: BlueNRG_LP-EVB Set-up :

  • TIM1_CH1 : PA4 BlueNRG_LPS-EVB Set-up :
  • TIM2 CH2 : PB4

Connect USART1 TX/RX to respectively RX and TX pins of PC UART (could be done through a USB to UART adapter) :

  • Connect BlueNRG_LP board USART1 TX pin to PC COM port RX signal
  • Connect BlueNRG_LP board USART1 RX pin to PC COM port TX signal
  • Connect BlueNRG_LP board GND to PC COM port GND signal

Launch serial communication SW on PC (as HyperTerminal or TeraTerm) with proper configuration :

  • 115200 bauds
  • 8 bits data
  • 1 start bit
  • 1 stop bit
  • no parity
  • no HW flow control

Detailed Description

How to use the HAL PKA API to generate an ECC signature.

KEIL project

To use the project with KEIL uVision 5 for ARM, please follow the instructions below:

  1. Open the KEIL uVision 5 for ARM and select Project->Open Project menu.
  2. Open the KEIL project C:\Users\{username}\ST\BlueNRG-LP_LPS DK x.x.x\Projects\Periph_Examples\HAL\PKA_PointCheck_IT\MDK-ARM\{STEVAL-IDB012V1}\PKA_PointCheck_IT.uvprojx
  3. Select desired configuration to build
  4. Select Project->Rebuild all target files. This will recompile and link the entire application
  5. To download the binary image, please connect an USB cable in your board (CMSIS-DAP upgrade).
  6. Select Project->Download to download the related binary image.
  7. Alternatively, open the BlueNRG-LP Flasher utility and download the built binary image.

IAR project

To use the project with IAR Embedded Workbench for ARM, please follow the instructions below:

  1. Open the Embedded Workbench for ARM and select File->Open->Workspace menu.
  2. Open the IAR project C:\Users\{username}\ST\BlueNRG-LP_LPS DK x.x.x\Projects\Periph_Examples\HAL\PKA\PKA_PointCheck_IT\EWARM\{STEVAL-IDB012V1}\PKA_PointCheck_IT.eww
  3. Select desired configuration to build
  4. Select Project->Rebuild All. This will recompile and link the entire application
  5. To download the binary image, please connect an USB cable in your board (CMSIS-DAP upgrade).
  6. Select Project->Download and Debug to download the related binary image.
  7. Alternatively, open the Flasher utility and download the built binary image.

WiSE-Studio project

To use the project with WiSE-Studio IDE (GCC toolchain), please follow the instructions below:

  1. Open the WiSE-Studio IDE
  2. Select File, Import, Existing Projects into Workspace C:\Users\{username}\ST\BlueNRG-LP_LPS DK x.x.x\Projects\Periph_Examples\HAL\PKA\PKA_PointCheck_IT\WiSE-Studio\{STEVAL-IDB012V1}
  3. Select desired configuration to build
  4. Select Project->Build Project. This will recompile and link the entire application
  5. To download the binary image, please connect an USB cable in your board (CMSIS-DAP upgrade).
  6. Select Run->Run/Debug to download the related binary image.
  7. Alternatively, open the Flasher utility and download the built binary image.

Usage

How to use the PKA peripheral to determine if a point is on a curve. This allows validating an external public key.

For this example, 2 test vector have been extracted from National Institute of Standards and Technology (NIST) - Cryptographic Algorithm Validation Program (CAVP) in order to demonstrate the usage of the hal. Those reference files can be found under: "http://csrc.nist.gov/groups/STM/cavp/documents/dss/186-3ecdsatestvectors.zip (ZIP PKV.rsp)"

Those two test vectors have been choosen to demonstrate the behavior in a case where the input point validates the curve equation and in a case where the input point does not verify the curve equation. Their definition is included in PKV.c. You can refer to this file for more informations.

The selected curve for this example is P-256 (ECDSA-256) published by NIST in Federal Information Processing Standards Publication FIPS PUB 186-4. The description of this curve is present in file Src/prime256v1.c.

In case of success, the LED2 is ON. In case of any error, the LED2 is toggling slowly.

BlueNRG_LP-EVB Set-up Connect USART1 TX/RX to respectively RX and TX pins of PC UART (could be done through a USB to UART adapter) :

  • Connect BlueNRG_LP board USART1 TX pin to PC COM port RX signal
  • Connect BlueNRG_LP board USART1 RX pin to PC COM port TX signal
  • Connect BlueNRG_LP board GND to PC COM port GND signal

Launch serial communication SW on PC (as HyperTerminal or TeraTerm) with proper configuration :

  • 115200 bauds
  • 8 bits data
  • 1 start bit
  • 1 stop bit
  • no parity
  • no HW flow control

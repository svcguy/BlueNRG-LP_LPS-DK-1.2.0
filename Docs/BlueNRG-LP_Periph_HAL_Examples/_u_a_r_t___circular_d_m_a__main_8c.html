Detailed Description

How to use the HAL UART API for reception in circular DMA mode.

KEIL project

To use the project with KEIL uVision 5 for ARM, please follow the instructions below:

  1. Open the KEIL uVision 5 for ARM and select Project->Open Project menu.
  2. Open the KEIL project C:\Users\{username}\ST\BlueNRG-LP_LPS DK x.x.x\Projects\Periph_Examples\HAL\UART\UART_CircularDMA\MDK-ARM\{STEVAL-IDB011V1|STEVAL-IDB012V1}\UART_CircularDMA.uvprojx
  3. Select desired configuration to build
  4. Select Project->Rebuild all target files. This will recompile and link the entire application
  5. To download the binary image, please connect an USB cable in your board (CMSIS-DAP upgrade).
  6. Select Project->Download to download the related binary image.
  7. Alternatively, open the BlueNRG-LP Flasher utility and download the built binary image.

IAR project

To use the project with IAR Embedded Workbench for ARM, please follow the instructions below:

  1. Open the Embedded Workbench for ARM and select File->Open->Workspace menu.
  2. Open the IAR project C:\Users\{username}\ST\BlueNRG-LP_LPS DK x.x.x\Projects\Periph_Examples\HAL\UART\UART_CircularDMA\EWARM\{STEVAL-IDB011V1|STEVAL-IDB012V1}\UART_CircularDMA.eww
  3. Select desired configuration to build
  4. Select Project->Rebuild All. This will recompile and link the entire application
  5. To download the binary image, please connect an USB cable in your board (CMSIS-DAP upgrade).
  6. Select Project->Download and Debug to download the related binary image.
  7. Alternatively, open the Flasher utility and download the built binary image.

WiSE-Studio project

To use the project with WiSE-Studio IDE (GCC toolchain), please follow the instructions below:

  1. Open the WiSE-Studio IDE
  2. Select File, Import, Existing Projects into Workspace C:\Users\{username}\ST\BlueNRG-LP_LPS DK x.x.x\Projects\Periph_Examples\HAL\UART\UART_CircularDMA\WiSE-Studio\{STEVAL-IDB011V1|STEVAL-IDB012V1}
  3. Select desired configuration to build
  4. Select Project->Build Project. This will recompile and link the entire application
  5. To download the binary image, please connect an USB cable in your board (CMSIS-DAP upgrade).
  6. Select Run->Run/Debug to download the related binary image.
  7. Alternatively, open the Flasher utility and download the built binary image.

Usage

How to use the HAL UART API for reception in circular DMA mode.

The UART peripheral configuration is ensured by the HAL_UART_Init() function. This later is calling the HAL_UART_MspInit() function which core is implementing the configuration of the needed UART resources according to the used hardware (CLOCK, GPIO, DMA and NVIC). You may update this function to change UART configuration. In this example, a DMA channel is associated to USART1 RX (Circular mode).

The UART/Hyperterminal communication is then initiated (Board is sending an information message, inviting user to enter characters on Terminal).

Application then starts the Reception process, using dedicated HAL UART API :

  • HAL_UART_Receive_DMA()

Notifications to application, that some data are available in reception buffer, are done through the execution of a user callback : This callback will be executed when any of following events occurs :

  • TC (Transfer Complete) : Rx buffer is full. (In case of Circular DMA, reception could go on, and next reception data will be stored in index 0 of reception buffer by DMA).

BlueNRG_LP-EVB board LEDs are used to monitor the transfer status:

  • LED2 turns ON if transmission/reception is complete and OK.
  • LED3 turns ON when when there is an error in transmission/reception process.

USARTx/UARTx instance used and associated resources can be updated in "UART_CircularDMA_main.h" file depending hardware configuration used.

When the parity is enabled, the computed parity is inserted at the MSB position of the transmitted data.

BlueNRG_LP-EVB Set-up Connect USART1 TX/RX to respectively RX and TX pins of PC UART (could be done through a USB to UART adapter) :

  • Connect BlueNRG_LP board USART1 TX pinto PC COM port RX signal
  • Connect BlueNRG_LP board USART1 RX pin to PC COM port TX signal
  • Connect BlueNRG_LP board GND to PC COM port GND signal

Launch serial communication SW on PC (as HyperTerminal or TeraTerm) with proper configuration :

  • 115200 bauds
  • 8 bits data
  • 1 start bit
  • 1 stop bit
  • no parity
  • no HW flow control

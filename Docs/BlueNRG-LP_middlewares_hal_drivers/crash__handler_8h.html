#define ASSERT_HANDLER (   expression)

Macro to store in RAM the register information where an assert is verified.

All the information stored are words (32 bit):

  • Assert Signature
  • SP

  • R1
  • R2
  • R3
  • R12
  • LR
  • PC
  • xPSR

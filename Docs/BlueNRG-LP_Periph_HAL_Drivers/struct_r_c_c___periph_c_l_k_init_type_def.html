uint32_t RCC_PeriphCLKInitTypeDef::RFClockSelection

RADIO IP does not use directly the system clock for its APB / AHB interfaces but the system clock with a potential divider (2 or 4). This parameter can be a value of RF RC64MPLL Clock Divider The supported configurations are:

  • CLK_SYS = 1 MHz / 2 MHz / 4 MHz / 8 MHz –> Not possible to use RADIO IP
  • CLK_SYS = 64 MHz / 32 MHz / 16 MHz –> Allow the use of the RADIO IP. The only constraint is that the CLK_SYS shall be bigger or equal than RADIO IP clk

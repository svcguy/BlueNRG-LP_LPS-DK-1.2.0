Detailed Description

RNG HAL module driver. This file provides firmware functions to manage the following functionalities of the Random Number Generator (RNG) peripheral:

Author
RF Application Team
  • Initialization and configuration functions
  • Peripheral Control functions
  • Peripheral State functions
==============================================================================
                   ##### How to use this driver #####
==============================================================================
[..]
    The RNG HAL driver can be used as follows:

    (#) Enable the RNG controller clock using __HAL_RCC_RNG_CLK_ENABLE() macro
        in HAL_RNG_MspInit().
    (#) Activate the RNG peripheral using HAL_RNG_Init() function.
    (#) Wait until the 16 bit Random Number Generator contains a valid
        random data using polling mode.
    (#) Get the 16 bit random number using HAL_RNG_GenerateRandomNumber() function.

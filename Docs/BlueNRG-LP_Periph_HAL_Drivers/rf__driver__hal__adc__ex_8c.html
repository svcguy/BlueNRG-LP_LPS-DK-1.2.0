Detailed Description

This file provides firmware functions to manage the following functionalities of the Analog to Digital Convertor (ADC) peripheral:

Author
RF Application Team
  • Operation functions ++ Start, stop, get result of conversions of ADC group injected, using 2 possible modes: polling, interruption. ++ Calibration +++ ADC automatic self-calibration +++ Calibration factors get or set
  • Control functions ++ Channels configuration on ADC group injected
  • State functions ++ ADC group injected contexts queue management Other functions (generic functions) are available in file "rf_driver_hal_adc.c".

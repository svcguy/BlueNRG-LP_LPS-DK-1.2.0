uint16_t  attr_handle,
ble_gatt_clt_write_ops_t *  wr_ops_p

This function is used to extract a prepare write from the queue.

Parameters
conn_handlethe connection handle from which the request is received.
attr_handleThe requested attribute handle to find. If it is set to 0x0000 then the first found prepare write entry will be returned.
[out]wr_ops_pThe prepare write pointer.
Returns

  • BLE_STATUS_ERROR: no prepare write request was found.

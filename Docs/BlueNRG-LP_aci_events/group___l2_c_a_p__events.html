)

Event generated when a request or a response is received from the peer to create an L2CAP channel.

Parameters
Connection_HandleConnection handle that identifies the connection. Values:
  • 0x0000 ... 0x0EFF
Event_TypeValues:
  • 0x00: L2CAP_CONN_REQ
  • 0x01: L2CAP_CONN_RESP
ResultResult of the connection request if Event_Type is L2CAP_CONN_RESP. To be ignored if Event_Type is L2CAP_CONN_REQ. Values:
  • 0x0000: L2CAP_CONN_SUCCESSFUL
  • 0x0002: L2CAP_SPSM_NOT_SUPPORTED
  • 0x0004: L2CAP_NO_RESOURCE_AVAILABLE
  • 0x0005: L2CAP_INSUFFICIENT_AUTHENTICATION
  • 0x0006: L2CAP_INSUFFICIENT_AUTHORIZATION
  • 0x0007: L2CAP_INSUFFICIENT_ENCRYPTION_KEY_SIZE
  • 0x0008: L2CAP_INSUFFICIENT_ENCRYPTION
  • 0x0009: L2CAP_INVALID_SOURCE_CID
  • 0x000A: L2CAP_SOURCE_CID_ALREADY_ALLOCATED
  • 0x000B: L2CAP_UNACCEPTABLE_PARAMETERS
IdentifierThis is the identifier which associates the request to the response.
SPSMSimplified Protocol/Service Multiplexer Values:
  • 0x0001 ... 0x00FF
CIDThe local channel endpoint that identifies the L2CAP channel. To be ignored if Event_Type is L2CAP_CONN_REQ.
Remote_CIDThe channel endpoint on the remote device.
Peer_MTUThe maximum SDU size (in octets) that the L2CAP layer entity sending the L2CAP_LE_CREDIT_BASED_CONNECTION_REQ can receive on this channel. Values:
  • 23 ... 65535
Peer_MPSThe maximum PDU payload size (in octets) that the L2CAP layer entity sending the L2CAP_LE_CREDIT_BASED_CONNECTION_REQ is capable of receiving on this channel. Values:
  • 23 ... 65535
Initial_CreditsThe number of K-frames that the peer device can send to the L2CAP layer entity as soon as the L2CAP channel is established.
Return values
None

The event is given by the L2CAP layer when a connection update request is received from the slave. The upper layer which receives this event has to respond by sending a aci_l2cap_connection_parameter_update_resp command.

Parameters
Connection_HandleHandle of the connection related to this L2CAP procedure.
IdentifierThis is the identifier which associates the request to the response.
L2CAP_LengthLength of the L2CAP connection update request.
Interval_MinMinimum value for the connection event interval. This shall be less than or equal to Conn_Interval_Max. Time = N * 1.25 msec. Values:
  • 0x0006 (7.50 ms) ... 0x0C80 (4000.00 ms)
Interval_MaxMaximum value for the connection event interval. This shall be greater than or equal to Conn_Interval_Min. Time = N * 1.25 msec. Values:
  • 0x0006 (7.50 ms) ... 0x0C80 (4000.00 ms)
Slave_LatencySlave latency for the connection in number of connection events. Values:
  • 0x0000 ... 0x01F3
Timeout_MultiplierDefines connection timeout parameter in the following manner: Timeout Multiplier * 10ms. Values:
  • 10 (100 ms) ... 3200 (32000 ms)
Return values
None

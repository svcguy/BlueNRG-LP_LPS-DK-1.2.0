BLE Stack Modular Configuration
  • BLE_STACK_BASIC_CONF: only basic features are enabled (e.g., Slave Only mode)
  • BLE_STACK_FULL_CONF: all features are enabled
  • BLE_STACK_SLAVE_DLE_CONF: Slave Only + Data Length Extension enabled
  • BLE_STACK_SLAVE_DLE_LE_2M_CODED_CONF: Slave Only + Data Length Extension + PHY 2M/Coded PHY enabled

Uncomment one of the lines or define the configuration in your toolchain compiler preprocessor.

Note
It is mandatory to define one (and only one) of the options.

BlueNRG-LP/BlueNRG-LPS Key Resources

This document contains the key resources to be referred/used in order to evaluate BlueNRG-LP, BlueNRG-LPS Bluetooth Low Energy (LE) Systems On Chips devices features, design and test Bluetooth LE applications scenarios and, then, move to the firmware application development and tests. It also describes the 2.4 GHz proprietary radio resources and how to evaluate and use them. Further, reference PCB guidelines with BOM, schematics, and gerber files are provided in order to allow user to design its BlueNRG-LP/BlueNRG-LPS PCB, validate it through a full set of bringing up tests and finalize its BlueNRG-LP/BlueNRG-LPS products (HW & SW components).

Follow the list of key steps to be followed:

1) BlueNRG-LP/BlueNRG-LPS Bluetooth LE devices evaluation

2) Design, Develop and Test Bluetooth LE applications using BlueNRG-LP/BlueNRG-LPS development kits

3) 2.4 GHz proprietary radio solutions

4) Design & Bringing Up custom BlueNRG-LP/BlueNRG-LPS PCB

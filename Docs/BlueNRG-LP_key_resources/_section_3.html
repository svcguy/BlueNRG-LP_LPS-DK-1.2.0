What is BlueNRG-LP/BlueNRG-LPS Radio low level driver ?

  • The BlueNRG-LP/BlueNRG-LPS devices integrate a RF transceiver compliant to the Bluetooth specification and to the standard national regulations in the unlicensed 2.4 GHz ISM band.
  • The associated Radio low level driver provides access to the BlueNRG-LP/BlueNRG-LPS 2.4 GHz proprietary radio in order to send and receive packets without using the Bluetooth link layer.
  • The 2.4 GHz proprietary radio examples are built on top of the Radio Low level driver and they can be used as reference examples for building other applications that use the BlueNRG-LP/BlueNRG-LPS 2.4 GHz proprietary radio.

First option: use BlueNRG-LP/BlueNRG-LPS Navigator

  • BlueNRG-LP/BlueNRG-LPS Navigator is an user friendly GUI which lets you directly download and run the selected 2.4 GHz proprietary radio prebuilt application binary image on the BlueNRG-LP/BlueNRG-LPS platforms without a SWD interface (it just requires a BlueNRG-LP/BlueNRG-LPS development kit connected to a PC USB port)
  • It also provides demo applications description (html files) and access to board configurations and demo application source code if needed.
  • BlueNRG-LP/BlueNRG-LPS Navigator is available on STSW-BNRGLP-DK SW package on Start menu folder, ST BlueNRG-LP_LPS DK x.x.x, BlueNRG-LP/BlueNRG-LPS Navigator.
  • User is just request to connect one or two BlueNRG-LP/BlueNRG-LPS board on PC USB ports and then select the specific 2.4 GHz proprietary radio demonstration application to be tested.

Second option: move to SW development environment starting from available 2.4 GHz proprietary radio examples

  • BlueNRG-LP/BlueNRG-LPS, STSW-BNRGLP-DK SW package provides a set of 2.4 GHz proprietary radio examples allowing to test several applications scenarios which can be easily selected and downloaded on a BlueNRG-LP/BlueNRG-LPS platform using the BlueNRG-LP/BlueNRG-LPS Navigator tool.
  • User can identify which available demonstration application is close to his specific needs and follows such steps:
    • Open its IDE toolchains and related demonstration application project (IAR EWARM for ARM, Keil MDK-ARM, WiSE-Studio GCC toolchains examples are available on STSW-BNRGLP-DK SW package)
    • Modify the available source files according to his needs and then download and test the built binary file on the selected platform.
    • Run the built application and play with it!

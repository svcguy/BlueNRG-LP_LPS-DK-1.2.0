uint16_t Extended_Advertising_Report_t_s::Event_Type

Event Type parameter. Bits from 0 to 4: 0 Connectable advertising 1 Scannable advertising 2 Directed advertising 3 Scan response 4 Legacy advertising PDUs used Bits 5-6 Data status: 00b = Complete 01b = Incomplete, more data to come 10b = Incomplete, data truncated, no more to come 11b = Reserved for future use Flags:

  • 0x0001: Connectable advertising
  • 0x0002: Scannable advertising
  • 0x0004: Directed advertising
  • 0x0008: Scan response
  • 0x0010: Legacy advertising PDUs used

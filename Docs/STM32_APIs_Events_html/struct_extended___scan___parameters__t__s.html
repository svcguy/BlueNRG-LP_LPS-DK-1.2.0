uint8_t Extended_Scan_Parameters_t_s::Scan_Type

The Scan_Type parameter specifies the type of scan to perform. 0: Passive Scanning. No scan request PDUs shall be sent. 1: Active Scanning. Scan request PDUs may be sent. Values:

  • 0x00: Passive Scanning
  • 0x01: Active Scanning

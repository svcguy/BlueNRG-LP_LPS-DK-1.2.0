uint16_t devDiscoveryS::scanInterval

Time interval from when the Controller started its last LE scan until it begins the subsequent LE scan. The scan interval should be a number in the range 0x0004 to 0x4000. This corresponds to a time range 2.5 msec to 10240 msec. For a number N, Time = N x 0.625 msec.

uint16_t devDiscoveryS::scanWindow

Amount of time for the duration of the LE scan. The scan window shall be less than or equal to scan interval. The scan window should be a number in the range 0x0004 to 0x4000. This corresponds to a time range 2.5 msec to 10240 msec. For a number N, Time = N x 0.625 msec.

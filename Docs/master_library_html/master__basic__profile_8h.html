uint32_t Master_Process ( void  )

This function needs to be called inside a main loop to run the Master state machines.

Return values
Thereturn value is a bitmask to signal the application if the Host micro can enter in power save or not. The bitmask format is:
  • 1 bit flag to signal if the micro can enter in power save or not. This is the LSB.
  • 1 bit flag to signal if the micro needs to enable the power save with a timeout.
  • 30 bits power save timeout, the latest MSB. These 30 bits are valid if the second bit is set.

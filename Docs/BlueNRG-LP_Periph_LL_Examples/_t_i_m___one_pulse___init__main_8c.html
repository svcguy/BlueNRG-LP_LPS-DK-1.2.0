KEIL project

To use the project with KEIL uVision 5 for ARM, please follow the instructions below:

  1. Open the KEIL uVision 5 for ARM and select Project->Open Project menu.
  2. Open the KEIL project C:\Users\{username}\ST\BlueNRG-LP_LPS DK x.x.x\Projects\Periph_Examples\LL\TIM\TIM_OnePulse_Init\MDK-ARM\{STEVAL-IDB011V1|STEVAL-IDB012V1}\TIM_OnePulse_Init.uvprojx
  3. Select desired configuration to build
  4. Select Project->Rebuild all target files. This will recompile and link the entire application
  5. To download the binary image, please connect an USB cable in your board (CMSIS-DAP upgrade).
  6. Select Project->Download to download the related binary image.
  7. Alternatively, open the BlueNRG-LP Flasher utility and download the built binary image.

Usage

Configuration of a timer to generate a positive pulse in Output Compare mode with a length of tPULSE and after a delay of tDELAY. This example is based on the BLUENRG_LP TIM LL API. The peripheral initialization uses LL unitary service functions for optimization purposes (performance and size).

The pulse is generated on OC1.

TIM1 generates a positive pulse of 50 us after a delay of 50 us. User push-button is used to start TIMx counter.

TIMx_CHx delay and pulse length are measured every time a pulse is generated.

TIMx is configured to generate a single pulse (timer counter stops automatically at the next update event UEV).

BlueNRG_LP-EVB Set-up Connect the following pins to an oscilloscope to monitor the different waveforms:

  • TIMx_CH4
  • LED2 Connect USART1 TX/RX to respectively RX and TX pins of PC UART (could be done through a USB to UART adapter) :
  • Connect BlueNRG_LP board USART1 TX pin to PC COM port RX signal
  • Connect BlueNRG_LP board USART1 RX pin to PC COM port TX signal
  • Connect BlueNRG_LP board GND to PC COM port GND signal

Launch serial communication SW on PC (as HyperTerminal or TeraTerm) with proper configuration :

  • 115200 bauds
  • 8 bits data
  • 1 start bit
  • 1 stop bit
  • no parity
  • no HW flow control

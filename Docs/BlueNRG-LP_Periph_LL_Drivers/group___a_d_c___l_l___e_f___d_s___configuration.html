uint32_t  Width,
uint32_t  Ratio 
)

Configure the width (in bit) and the ratio of the output data from the Down Sampler.

Parameters
ADCxADC instance
WidthThis parameter can be one of the following values:
  • LL_ADC_DS_DATA_WIDTH_12_BIT (default)
  • LL_ADC_DS_DATA_WIDTH_13_BIT
  • LL_ADC_DS_DATA_WIDTH_14_BIT
  • LL_ADC_DS_DATA_WIDTH_15_BIT
  • LL_ADC_DS_DATA_WIDTH_16_BIT
RatioThis parameter can be one of the following values:
  • LL_ADC_DS_RATIO_01 (no down sampling, default)
  • LL_ADC_DS_RATIO_02
  • LL_ADC_DS_RATIO_04
  • LL_ADC_DS_RATIO_08
  • LL_ADC_DS_RATIO_16
  • LL_ADC_DS_RATIO_32
  • LL_ADC_DS_RATIO_64
  • LL_ADC_DS_RATIO_128
Return values
None

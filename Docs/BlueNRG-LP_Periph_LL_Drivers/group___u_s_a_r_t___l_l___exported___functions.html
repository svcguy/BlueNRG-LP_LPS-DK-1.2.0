void LL_FLASH_PageProtection ( FLASH_TypeDef *  FLASHx,
uint32_t  pageProt_0,
uint32_t  pageProt_1,
FunctionalState  state 
)

Page Protection function. It allows protecting from write and page erase by group of 8 pages.

Parameters
FLASHxFLASH Instance
pageProt_0It is a Bitfield built as follows:
  • bit[6:0] = SIZE: number of 2 kB pages to protect including the starting pagepageProt_0 param (provided in the offset bits)
  • bit[7]: reserved
  • bit[14:8] = OFFSET: page number to start the write protection (value between 0 and 0x7F)
  • bit[15]: reserved
  • bit[22:16] = SIZE: same meaning of bit[6:0]
  • bit[23]: reserved
  • bit[30:24] = OFFSET: same meaning of bit[22:16]
  • bit[31]: reserved
pageProt_1Same meaning of pageProt_0 param
statethis parameter can be: ENABLE DISABLE
Return values
none

Detailed Description

Return the frequency of the system clock device.

Note
If SYSCLK source is HSI, function returns values based on HSI_VALUE(**)
If SYSCLK source is HSE, function returns values based on HSE_VALUE(***)
(**) HSI_VALUE is a constant defined in this file (default value 64 MHz) but the real value may vary depending on the variations in voltage and temperature.
(***) HSE_VALUE is a constant defined in this file (default value 32 MHz). The HSE_VALUE value is divided from a 64 MHz PLL to return frequency of the crystal used.

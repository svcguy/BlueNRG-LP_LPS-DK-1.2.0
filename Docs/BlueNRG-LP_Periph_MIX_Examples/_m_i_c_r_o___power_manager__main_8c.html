Detailed Description

This is a demo that shows how to configure BlueNRG-LP power modes in order to save power.

KEIL project

To use the project with KEIL uVision 5 for ARM, please follow the instructions below:

  1. Open the KEIL uVision 5 for ARM and select Project->Open Project menu.
  2. Open the KEIL project C:\Users\{username}\ST\BlueNRG-LP_LPS DK x.x.x\Projects\Periph_Examples\MIX\MICRO\MICRO_PowerManager\MDK-ARM\{STEVAL-IDB011V1|STEVAL-IDB012V1}\MICRO_PowerManager.uvprojx
  3. Select desired configuration to build
  4. Select Project->Rebuild all target files. This will recompile and link the entire application
  5. To download the binary image, please connect an USB cable in your board (CMSIS-DAP upgrade).
  6. Select Debug->Start/Stop Debug Session to download the related binary image.
  7. Alternatively, open the Flasher utility and download the built binary image.

To use the project with IAR Embedded Workbench for ARM, please follow the instructions below:

  1. Open the Embedded Workbench for ARM and select File->Open->Workspace menu.
  2. Open the IAR project C:\Users\{username}\ST\BlueNRG-LP_LPS DK x.x.x\Projects\Periph_Examples\MIX\MICRO\MICRO_PowerManager\EWARM\{STEVAL-IDB011V1|STEVAL-IDB012V1}\MICRO_PowerManager.eww
  3. Select desired configuration to build
  4. Select Project->Rebuild All. This will recompile and link the entire application
  5. To download the binary image, please connect an USB cable in your board (CMSIS-DAP upgrade).
  6. Select Project->Download and Debug to download the related binary image.
  7. Alternatively, open the Flasher utility and download the built binary image.

Jumper settings

STEVAL-IDB01xV1
Jumper nameDescription
JP1It provides the voltage to the BlueNRG-LP circuit. It must be fitted. It can be used for current measurements of the BlueNRG-LP device.
JP2It is a switch between two power domains. BAT position: to provide power from battery holder; USB position: to provide power from USB connector.
JP3It connects the BLE_SWCLK pin of the BlueNRG-LP with the SWCLK pin of the USB_CMSISDAP. It must be fitted.
JP4It connects the BLE_SWDIO pin of the BlueNRG-LP with the SWDIO pin of the USB_CMSISDAP. It must be fitted.
JP5It connects the BLE_RSTN pin of the BlueNRG-LP with the rest of the board (the USB_CMSISDAP and RESET push button). It must be fitted.

Usage

This test provides an example for the BlueNRG-LP power modes. The power modes tested are SHUTDOWN LEVEL, POWER_SAVE_LEVEL_STOP_WITH_TIMER, POWER_SAVE_LEVEL_NOTIMER and POWER_SAVE_LEVEL_CPU_HALT.

  • The POWER_SAVE_MODE_SHUTDOWN turns off BlueNRG-LP device. The only wakeup source is a low pulse on the RSTN pad (no smart power management is in place, because there is no memory retention in this power save level configuration).
  • The POWER_SAVE_LEVEL_STOP_WITH_TIMER puts the BlueNRG-LP in deep sleep and the timer clock sources (LSI or LSE) remain running. Wakeup is possible from all IOs enabled for this scope, RTC, IWDG, LPUART (BlueNRG-LPS), radio IP and the Hal Virtual Timers.
  • The POWER_SAVE_LEVEL_NOTIMER puts the BlueNRG-LP in in deep sleep. All the peripherals and clock sources are turned off. Wakeup is possible only from IOs enabled for this scope.
  • The POWER_SAVE_LEVEL_CPU_HALT puts the BlueNRG-LP in CPU halt. Only the CPU is halted. The rest of the chip continues running normally. The chip will wake from any interrupt.

The demo supports some user commands:

  • s: SHUTDOWN LEVEL : the only wakeup source is a low pulse on the RSTN pad
  • t: POWER_SAVE_LEVEL_STOP_WITH_TIMER : wake on UART RX pin/timeout 5 sec (VTIMER)/button PUSH1 (PA10)
  • z: POWER_SAVE_LEVEL_STOP_WITH_TIMER : wake on UART RX pin/timeout 5 sec (RTC)/button PUSH1 (PA10)
  • u: POWER_SAVE_LEVEL_STOP_WITH_TIMER : wake on LPUART (BlueNRG-LPS)/button PUSH1 (PA10)
  • n: POWER_SAVE_LEVEL_NOTIMER : wake on UART RX pin/button PUSH1 (PA10)
  • c: POWER_SAVE_LEVEL_CPU_HALT : wake on button PUSH1 (PA10)
  • l: Toggle led DL1.
  • p: Print Hello World message.
  • r: Reset the BlueNRG-LP
  • ?: Display this help menu.
  • PUSH1 button: toggle led DL1.

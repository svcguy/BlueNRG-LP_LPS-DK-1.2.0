BlueNRG-LP/BlueNRG-LPS Development kits (STSW-BNRGLP-DK)
Release Notes

This document contains the release notes of the BlueNRG-LP, BlueNRG-LPS DK Development kits (STSW-BNRGLP-DK).

  • STSW-BNRGLP-DK SW package provides the software drivers, library and examples for the BlueNRG-BLE Stack Library v3.x (STMicrolectronics Bluetooth Low Energy stack), 2.4 GHz proprietary radio solutions and all the other device peripherals (ADC, DMA, GPIO, I2C, TIM, RCC,RNG, RTC, SPI, UART, IWDG, ...). It also includes the BlueNRG-LP/BlueNRG-LPS Navigators, the BlueNRG-X Radio Init Wizard and the Secure Bootloader PC applications for Windows.
  • This document describes the new features, changes, solved issues and limitations associated to each new release of the SW package.
  • A PC running Windows 10 operating systems (64 bits version), with available USB ports is required for installing the BlueNRG-LP, BlueNRG-LPS DK SW package.

Introduction

This document describes the migration steps from Bluetooth Low Energy stack v2.x to v3.x in terms of new APIs, events callbacks and new initialization parameters.
Further, it describes the main and most used APIs, events callbacks and parameters which have been modified/added from Bluetooth LE stack v2.x to v3.x. For detailed information about each API, event callback and related parameters please refer to the Bluetooth LE stack v3.x APIs, events callbacks doxygen documentation.

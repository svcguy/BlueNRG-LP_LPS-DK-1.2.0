Bluetooth LE stack v3.x error codes
  • Bluetooth LE stack v3.x error code policy is as follows (refer to file ble_status.h ):
    • The Standard Error Codes, as specified by the Bluetooth Specifications Vol.2, Part D ("Error Codes")) and related to the Controller, are the ones with prefix BLE_ERROR (range from 0x00 to 0x7F).
    • The Generic/System error codes reported by the different Protocol layers (Host), are the ones with prefix BLE_STATUS (they start with 0x80).
    • Some GAP layer APIs could return Controller error codes (prefix BLE_ERROR) since they directly interface to the Controller.

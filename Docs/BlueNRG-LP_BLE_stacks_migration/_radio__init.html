NOTEs:

  • (1) Bluetooth LE v2.x BlueNRG_Stack_Initialization_t structure has been renamed as BLE_STACK_InitParams on Bluetooth LE stack v3.x
  • (2) CTE_MaxNumAntennaIDs and CTE_MaxNumIQSamples are used for BlueNRG-LPS Direction Finding features.
  • BLE_STACK_InitParams parameters can be properly configured using the BlueNRG-LP/BlueNRG-LPS Radio Init Wizard PC tool.
